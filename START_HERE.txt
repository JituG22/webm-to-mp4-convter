🐛 COMMON ISSUES & QUICK FIXES

"Cannot connect to server"
  → Check if backend is running: lsof -i :5000
  → Start backend: cd server && npm start

"Port already in use"
  → Find process: lsof -i :3000 or lsof -i :5000
  → Kill it: kill -9 [PID]

"Conversion fails"
  → Check if file is valid WebM format
  → Look at server console for error message

"FFmpeg not found"
  → It's installed at: /opt/homebrew/bin/ffmpeg
  → Or install: brew install ffmpeg

More help in: SETUP.md (Troubleshooting section)
